## 🚀 كيفية البدء الآن

### **الخطوة 1: نسخ المشروع**
```bash
cd c:\Users\ahmed\Dev
# المشروع موجود بالفعل!
```

### **الخطوة 2: القراءة السريعة**
```bash
# اختر أحد هذه الملفات:
# - INDEX.md (للفهرسة)
# - QUICKSTART.md (للبدء السريع)
# - QUICK_REFERENCE.md (مرجع سريع)
```

### **الخطوة 3: التثبيت**
```bash
npm install
```

### **الخطوة 4: إعداد قاعدة البيانات**
```bash
# عدّل DATABASE_URL في backend/.env
npm run prisma:migrate --workspace=backend
```

### **الخطوة 5: التشغيل**
```bash
# Terminal 1
npm run dev --workspace=backend

# Terminal 2
npm run dev --workspace=frontend

# افتح http://localhost:3000
```
