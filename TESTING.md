# LeaveFlow - اختبار الهياكل

## جدول الاختبار

### 1. المصادقة والتفويض
- [ ] تسجيل مستخدم جديد كموظف
- [ ] تسجيل مستخدم جديد كمشرف
- [ ] تسجيل مستخدم جديد كرئيس قسم
- [ ] تسجيل الدخول بنجاح
- [ ] رفض تسجيل الدخول برقم سري خاطئ
- [ ] التحقق من صلاحيات الأدوار المختلفة

### 2. إدارة المستخدمين
- [ ] عرض الملف الشخصي للموظف
- [ ] تحديث بيانات الملف الشخصي
- [ ] عرض جميع المستخدمين (رئيس قسم)
- [ ] عرض موظفي الموقع (مشرف)
- [ ] ترقية موظف لمشرف
- [ ] تعطيل مستخدم

### 3. واجهة المستخدم
- [ ] تصميم مريح وسهل الاستخدام
- [ ] استجابة جيدة على أجهزة مختلفة
- [ ] نظام تنبيهات فعال
- [ ] تحميل سريع للصفحات

## ملاحظات إضافية
- تأكد من أن متغيرات البيئة صحيحة
- تأكد من اتصال قاعدة البيانات
- اختبر على متصفحات مختلفة
